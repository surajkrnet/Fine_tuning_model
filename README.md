📌 Overview
A Streamlit-based web application designed to streamline the fine-tuning process for Generative AI models. This platform provides an intuitive, interactive interface for uploading datasets, configuring model hyperparameters, and managing training workflows without needing to write complex scripts.

 ✨ Features
* **Interactive Web Interface:** Built entirely with Streamlit for a seamless user experience.
* **Dataset Management:** Easily upload, preview, and format training data (JSONL, CSV, etc.).
